What's the purpose of App?

This app is created by Ztric India team, to help the users of Electronic Vehicle.

User's with this App can easily access the charging station to get charged their vehicles.

How to use this App?

User with this App can easily find the nearby stations which are active, and using the QR code available on the satation they can connect by scanning the code.

User can find the stations within the specific radius from the current location.

Plug and Play.

This app provides the feature to the user that once plug the cable to station, station cares the further that cable might not get into unauthorised user, as station has Locking feature.

Once user done with the charging and pay the Bill amount, than the cable is unlocked from the station.
